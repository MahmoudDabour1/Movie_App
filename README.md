# Movies_With_Clean_Architecture

A movie app made in Flutter with clean architecture using BLoC and the movie db API..

## Features

* Search feature
* Now playing movies
* Popular movies
* Top rated movies
*  Movie details
* Movie cast
* Movie reviews
* Similar movies
*  On air tv shows
* Popular tv shows
* Top rated tv shows
* TV show details
* Similar tv shows
* TV show season details

## Installation

### 1. Clone the repo

```
git clone https://github.com/MahmoudDabour1/Movie_App.git
cd flutter-movie-app
```

### 2. Run pub get

```
flutter pub get
```
### 3. Run the app

```
flutter run
```
